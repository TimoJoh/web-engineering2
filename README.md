## Frontend starten 
1. cd .\frontend\
2. npm install
3. npm start

## Backend starten
1.  Maven Projekt initialisieren
    - pom.xml -> rechts click -> maven -> sync project
2. backend/src/main/java/com.weatherApp.WeatherWeb.api/
3. WeatherWebApplication starten
